# 📦 配件庫存管理系統

多門市配件補貨水位管理工具。

## 🌐 使用方式

**HTML 管理工具（所有門市共用）：**
https://snowsepch.github.io/accessory-inventory/

每個門市上傳各自的 XLS 資料，系統在瀏覽器中處理，資料不會上傳至伺服器。

## 📅 各門市行事曆訂閱

| 門市 | 訂閱連結 |
|------|---------|
| 門市01 | `webcal://raw.githubusercontent.com/snowsepch/accessory-inventory/main/calendars/store-01.ics` |

### iPhone 訂閱步驟：
1. 點擊上方連結（在 iPhone Safari 開啟）
2. 彈出「加入行事曆」→ 確認
3. 之後每次更新 .ics 後，iPhone 24 小時內自動同步

## 🔄 更新行事曆

在 HTML 工具計算庫存水位後，點「🔄 更新行事曆」按鈕即可自動更新。
